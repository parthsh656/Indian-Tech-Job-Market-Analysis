# Indian-Tech-Job-Market-Analysis
End-to-end data pipeline analyzing 5,000 Indian tech job postings (2024–2026). Built with Python (Pandas, Seaborn), it maps hiring trends, skill demands, and salary curves. Highlights a strict 10x seniority pay gap, top-tier AI skill premiums, and a high applicant-to-opening friction ratio of 83:1. Built for optimized analytics translation.
